# Tic Tac Toe

> [!NOTE]
> A fun, simple project built for practice and experimentation.

![Screenshot](/assets/images/screenshot.png)

A simple, interactive Tic-Tac-Toe game implemented using JavaScript. This project leverages Factory Functions, the Module Pattern, and Immediately Invoked Function Expressions (IIFEs) to structure the code in a modular and clean way.
